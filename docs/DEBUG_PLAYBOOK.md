## Common Failure Patterns

| Symptom | First Check | Likely Cause |
|---------|------------|---------------|
| Builder not scaffolding | Config | Missing or invalid config |
| Agent not responding | API key | Missing key |
| Output malformed | Template | Template syntax error |
| File generation fails | Permissions | Write permission issue |
